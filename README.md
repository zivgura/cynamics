## Installation
Download the repository
###### Server side:
1. Install npm modules:
```bash
 cd server
 npm install
```
2. Start up the server: 
```bash
 cd server
 npm run start-sever
```
###### Client Side:
1. Install npm modules: 
```bash
 cd client
 npm install
```
2. Start up the client:
```bash
 cd client
 npm run start-client
```

## Usage
Go to http://localhost:8080
